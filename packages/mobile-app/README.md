# Mobile App - اینفرنال

Nuxt 3 + Vue 3 + Capacitor mobile application for the Puzzle Minigames.

## Features

- **Nuxt 3 & Vue 3**: Modern reactive framework with Composition API
- **SCSS Styling**: Organized styles with variables and nesting
- **Three.js**: 3D graphics and animations
- **Matter.js**: Physics engine for mini-games
- **Capacitor**: Native mobile deployment
- **RTL Support**: Persian (Farsi) language interface

## Development

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Generate Static Site
```bash
npm run generate
```
